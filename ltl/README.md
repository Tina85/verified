# LTL

**L**inear **T**emporal **L**ogic in Coq, implemented with `CoInductive`.

## Ingredients

* Atomic
* Not, And, Or
* Ne**x**t
* **U**ntil
* Always (**g**lobally)
* Eventually (**f**uture)
* **R**elease
* ... and some of their properties and alternative definitions.

## Note

Asserts classical logic (`P \/ ~P`).
